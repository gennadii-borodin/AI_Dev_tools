## Overview

The –ú–∏–Ω–∏-–∞–Ω–∫–µ—Ç–∞ application is a simple web-based survey system that allows users to fill out questionnaires and submit their responses. The application consists of a FastAPI backend that serves both the API endpoints and the frontend HTML/JS interface.

## System Components

### 1. Backend (FastAPI)
- **Framework**: FastAPI with Uvicorn ASGI server
- **Language**: Python 3.12
- **Responsibilities**:
  - Serve the frontend HTML/JS files
  - Provide REST API endpoints for questions and answers
  - Store user responses in memory

### 2. Frontend (HTML/JS)
- **Technologies**: HTML5, CSS3, Vanilla JavaScript
- **Responsibilities**:
  - Display the survey form
  - Load questions from the backend
  - Collect user responses
  - Submit answers to the backend
  - Show confirmation message after submission

### 3. Data Storage
- **Type**: In-memory dictionary storage
- **Structure**: Dictionary of dictionaries mapping usernames to their answers
